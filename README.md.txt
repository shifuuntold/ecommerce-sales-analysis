# E-Commerce Sales Analysis Dashboard

## Project Overview

This project analyzes e-commerce sales transactions from 2022–2024 using Microsoft Excel.

The objective was to identify revenue trends, profitability drivers, category performance, and regional sales patterns through interactive dashboard reporting.

The dashboard was built using Pivot Tables, Pivot Charts, KPI Cards, formulas, and slicers.

## Dataset Information

The dataset contains 3,500 e-commerce transactions and includes:

* Order Date
* Product Name
* Category
* Region
* Quantity
* Sales
* Profit

Additional calculated fields:

* Year
* Month
* Quarter
* Profit Margin %

## Key Insights

### Category Performance

* Electronics generated the highest revenue (5.33M).
* Electronics generated the highest profit (923K).

### Regional Performance

* West was the highest-performing region.
* North generated the lowest revenue.

### Year-over-Year Trends

| Year | Sales |
| ---- | ----: |
| 2022 | 3.26M |
| 2023 | 3.79M |
| 2024 | 3.63M |

* Revenue increased by 16.3% from 2022 to 2023.
* Revenue decreased slightly by 4.3% from 2023 to 2024.

### Quarterly Trends

| Quarter | Sales |
| ------- | ----: |
| Q1      | 2.61M |
| Q2      | 2.70M |
| Q3      | 2.66M |
| Q4      | 2.71M |

* Q4 recorded the highest revenue.
* Sales remained relatively stable across all quarters.

---

## Business Recommendations

1. Continue investing in Electronics due to strong revenue and profit performance.
2. Investigate growth opportunities in the North region.
3. Monitor the decline observed in 2024 sales performance.
4. Use strong Q4 performance patterns when planning promotions and inventory.
